| **Proposed Project Title**  | Gender, Racial, and Ethnic Differences in Suicide Rates Across Age Groups                                                                                                                                  |
| **Question**                     | How do suicide rates differ between males and females across racial and ethnic backgrounds, and are there specific age groups within these populations at higher risk?  |
